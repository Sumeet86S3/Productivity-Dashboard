# Productivity Dashboard

A full-stack MERN application to manage tasks, notes, and goals.

## Tech Stack
- React (Vite)
- Node.js + Express
- MongoDB Atlas
- Tailwind CSS

## Features
- User Authentication (JWT)
- Task Management
- Notes System
- Goal Tracking

## Setup

### Backend
cd server
npm install
npm start

### Frontend
cd client
npm install
npm run dev
